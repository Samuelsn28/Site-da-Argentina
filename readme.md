
# O site da Argentina

## Objetivo

Expor informações importantes sobre a Argentina.

## Telas
- Home: com informações sobre o país;
- Me financie: para doação;
- Galeria: fotos do país.

## Framework

Foi usado o framework JavaScript Materialize.

## Linguagens

**Front-end:** HTML, CSS e JavaScript.
**Back-end:** nada

## Cronologia

O site foi produzido em 2 horas.
